# voting-system-python

## Project Overview

This project is a simple Voting System built using Python.  
It allows users to vote for candidates and view the voting results.

The program stores votes in a dictionary and determines the winner based on the highest number of votes.

## Features

- Vote for candidates (A, B, C)  
- Display vote count for each candidate  
- Automatically determine the winner  
- Simple menu-driven interface  

## Technologies Used

- Python  

## How the Program Works

### 1. Voting

- User selects option to vote  
- Enters candidate name (A/B/C)  
- Vote count increases  


### 2. Show Results

- Displays votes for each candidate  

Example:

A: 2 votes
B: 1 votes
C: 0 votes

- Displays winner based on highest votes  

Example:

Winner: A

### 3. Exit

- Stops the program  

## How to Run

1. Install Python  
2. Save file as `voting_system.py`  
3. Run the program: voting_system.py
